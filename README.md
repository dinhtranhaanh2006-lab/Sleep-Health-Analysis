# Sleep Health Analysis

## Overview

This project analyzes the relationship between sleep duration, gender, and stress levels using statistical methods in Python.

---

## Objectives

- Compare average sleep duration between males and females.
- Analyze the relationship between stress level and sleep duration.
- Apply statistical analysis to support data-driven conclusions.

---

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Google Colab

---

## Skills Applied

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Independent Samples T-Test
- Correlation Analysis
- Data Visualization

---

## Dataset

Sleep Health and Lifestyle Dataset
